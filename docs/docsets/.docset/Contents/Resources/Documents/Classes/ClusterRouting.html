ClusterRouting

class ClusterRouting

The ClusterRouting calculates the routing between clusters. The function getCompleteSchedule will return the full schedule given the input clusters, working hour and the current station.

To do

The class should be able to handle different working hour for different days. Also the last repeat should be considered when calling getCompleteSchedule.

  • getCompleteSchedule(info:workingHour:currentStat:)

    Caculate the full schedule with the clusterInfo, workingHour and station current at

    Steps

    1. Check the time doesn’t exceed working hour limit
    2. Go to nearest cluster
    3. Cut the cluster if you can not visit all station in last cluster
    4. Finish a day
    5. Go back to step 1 for another day
